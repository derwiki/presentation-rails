## Includes:

  - `reveal.js`: JS slideshow library
  - `highlight.js`: JS syntax highlighting library, with themes

## Usage:

1. Make sure the prerequisites are in place:

        bundle

1. Build the presentation and open it:

        bundle exec rake
        open index.html
